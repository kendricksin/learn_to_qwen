# Customer Registration Application

This is a simple customer registration application built with Python and SQLite.

## Files in this project:

- `customer_app.py` - Main application code
- `customers.db` - SQLite database file (created automatically)
- `requirements.txt` - Required packages (none needed for this app)
- `README.md` - Project documentation and tutorial
- `test_app.py` - Test script to verify application functionality
- `demo.txt` - Usage demonstration

## How to run:

1. Make sure you have Python 3 installed
2. Download all files to a folder
3. Run the application: `python3 customer_app.py`
4. Follow the on-screen prompts

## Features:

- Add new customers with name, email, and phone
- View all registered customers
- Search for customers by name or email
- Prevents duplicate email entries
- Stores data persistently in SQLite database

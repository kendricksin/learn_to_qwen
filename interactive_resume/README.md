# Interactive Resume Project - AI Prompting Guide

Learn how to effectively prompt AI to help you build a modern, interactive resume web page using React and the JSON Resume standard.

## Tutorial Steps

Follow these sequential steps to build your project from scratch:

1. **[Step 1: Fill out your plan.md](STEP_1.md)**
   - Define your goals and prepare a structured plan for the AI.
2. **[Step 2: Paste it into Qwen Chat](STEP_2.md)**
   - Start your conversation with the AI using your plan.
3. **[Step 3: Iterate on the Conversation](STEP_3.md)**
   - Refine the design, handle complex components, and ask for clarifications.
4. **[Step 4: Build the Project](STEP_4.md)**
   - Finalize your web app and check the reference answers.

---

> [!TIP]
> Each step contains navigation links at the bottom to guide you through the process.
